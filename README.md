## How to run

1. Clone this repository
   
   ```sh
   https://github.com/ProgrammingHeroWC4/review-website-rownok420.git
   ```

2. Run
   
    ```sh
    npm install
    ```
   
3. Then run to see project in live server
   
   ```sh
   npm start
   ```
   
4. This app should be available in http://localhost:3000/
